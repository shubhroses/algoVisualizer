Selection Sort

Description

Selection Sort is another elementary sorting algorithm that's both intuitive to understand and straightforward to implement. It's named for its approach to sorting: it repeatedly "selects" the smallest (or largest) element from the unsorted portion of the list and swaps it to its final position in the sorted list.

How it Works:

  1. Initialize: Start at the first element in the list and treat it as the minimum.
  2. Scan: Scan through the remaining elements to find if there is an element smaller than your current minimum.
  3. Swap: If a smaller element is found, swap it with the minimum.
  4. Increment: Move to the next element and repeat the process until the array is sorted.

After each pass, the smallest remaining element is placed at the front of the remaining list, and the 'unsorted' section of the list becomes smaller.

Key Features:

  • Not Stable: Unlike Bubble Sort, Selection Sort is not a stable sorting algorithm, meaning that the relative order of equal elements may change during the sorting process.
  • Not Adaptive: The algorithm does not adapt to the existing order of the elements. Therefore, it will make the same number of comparisons regardless of the initial arrangement of the elements, leading to a time complexity of O(n2) in all cases.
  • In-Place Sorting: Like Bubble Sort, Selection Sort is also an in-place sorting algorithm and doesn't require additional memory for sorting, making its space complexity O(1).

In summary, while Selection Sort is easy to understand and implement, it's generally not suitable for large datasets due to its quadratic time complexity. However, it can be useful in scenarios where memory is a constraint, as it doesn't require additional storage.

Time Complexity: O(n2) for worst-case, average-case, and best-case.

    function selectionSort(arr) {
        let n = arr.length;
        for (let i = 0; i < n - 1; i++) {
            let minIndex = i;
            for (let j = i + 1; j < n; j++) {
                if (arr[j] < arr[minIndex]) {
                    minIndex = j;
                }
            }
            [arr[i], arr[minIndex]] = [arr[minIndex], arr[i]];
        }
    }

    public void selectionSort(int arr[]) {
        int n = arr.length;
        for (int i = 0; i < n - 1; i++) {
            int minIndex = i;
            for (int j = i + 1; j < n; j++) {
                if (arr[j] < arr[minIndex]) {
                    minIndex = j;
                }
            }
            int temp = arr[minIndex];
            arr[minIndex] = arr[i];
            arr[i] = temp;
        }
    }
