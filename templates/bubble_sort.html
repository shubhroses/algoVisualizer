Bubble Sort

25 Bars

Description

Bubble Sort is one of the simplest sorting algorithms, commonly used for educational purposes. Its name comes from the way smaller or larger elements "bubble" to the top or bottom of the list during the sorting process.

How it Works:

  1. Iterate: The algorithm iterates through the list from the beginning to the end, comparing adjacent elements.
  2. Swap: If the adjacent elements are in the wrong order, they are swapped.
  3. Repeat: This process is repeated for multiple passes through the list until no more swaps are needed, indicating that the list is sorted.

The algorithm can be optimized by noticing that the largest element "bubbles up" to the last position on the first pass, the second largest on the second pass, and so on. This means that each subsequent pass can consider one less element.

Key Features:

  • Stable Sort: Similar to Merge Sort, Bubble Sort is a stable sorting algorithm, meaning that elements with equal values remain in the same relative order after sorting.
  • Adaptive: While Bubble Sort has a worst-case and average-case time complexity of O(n2), it can recognize an already sorted list and terminate early, making its best-case time complexity O(n).
  • In-Place Sorting: Bubble Sort is an in-place sorting algorithm, meaning it doesn't require any additional memory for sorting. Therefore, its space complexity is O(1).

In summary, Bubble Sort is easy to understand and implement but generally inefficient for large datasets. It is best suited for educational purposes or for small lists where its simplicity is an advantage.

function bubbleSort(arr) {
    let n = arr.length;
    for (let i = 0; i < n - 1; i++) {
        for (let j = 0; j < n - i - 1; j++) {
            if (arr[j] > arr[j + 1]) {
                [arr[j], arr[j + 1]] = [arr[j + 1], arr[j]];
            }
        }
    }
}

public void bubbleSort(int arr[]) {
    int n = arr.length;
    for (int i = 0; i < n - 1; i++) {
        for (int j = 0; j < n - i - 1; j++) {
            if (arr[j] > arr[j + 1]) {
                int temp = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = temp;
            }
        }
    }
}
